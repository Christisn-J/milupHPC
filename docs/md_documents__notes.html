N-Body & SPH

  • N-Body:
    • (parallel) tree construction
      • bounding box
      • domain list
      • tree
      • common coarse tree
    • (parallel) pseudo-particles
    • (parallel) gravitational force
  • SPH with N-Body:
    • (parallel) tree construction
      • bounding box
      • domain list
      • tree
      • common coarse tree
    • (parallel) pseudo-particles
    • (parallel) gravitational force
    • (parallel) SPH forces
  • SPH without N-Body:
    • (parallel) tree construction
      • bounding box
      • domain list
      • tree
      • common coarse tree
    • (parallel) SPH forces

Thus 4 modules:

  • (parallel) tree construction
  • (parallel) pseudo-particles
  • (parallel) gravitational force
  • (parallel) SPH forces
